## Aperçu

Tous les modèles d'IA dans Cattus ont un accès natif à Internet, leur permettant de rechercher sur le web en temps réel pendant vos conversations. Cette capacité garantit que vous recevez des informations à jour, des événements actuels et les dernières données sans quitter l'interface de chat.

## Comment ça fonctionne

### Recherche Web Automatique

<Steps>
  <Step title="Posez Votre Question">
    Posez simplement des questions nécessitant des informations actuelles ou des données en temps réel.
  </Step>

  <Step title="Le Modèle Recherche Automatiquement">
    Le modèle d'IA détecte quand une recherche web est nécessaire et recherche automatiquement des informations pertinentes sur Internet.
  </Step>

  <Step title="Obtenez des Réponses Actuelles">
    Recevez des réponses basées sur les dernières informations disponibles en ligne, avec des sources si applicable.
  </Step>
</Steps>

## Avantages

### Intégration Native

La recherche web est intégrée directement dans tous les modèles, ce qui signifie :

- **Aucune Étape Supplémentaire** : Vous n'avez rien à activer ou configurer
- **Expérience Fluide** : Le modèle décide quand rechercher en fonction de votre requête
- **Accès Universel** : Disponible sur tous les modèles d'IA dans Cattus
- **Toujours Actuel** : Obtenez les informations les plus récentes disponibles en ligne

## Bonnes Pratiques

### Formuler des Requêtes Efficaces

Pour obtenir les meilleurs résultats de la recherche web :

1. **Soyez Précis** : Incluez des détails pertinents comme des dates, lieux ou sujets spécifiques
2. **Indiquez la Récence** : Utilisez des expressions comme "dernier," "actuel," ou "récent" si approprié
3. **Posez Directement** : Formulez les questions naturellement comme vous le feriez lors d'une recherche en ligne
4. **Demandez des Sources** : Demandez au modèle de fournir des sources lorsque vous devez vérifier les informations

### Exemples

<CodeGroup>

```text Bonne Requête
Quelles sont les dernières fonctionnalités annoncées pour Python 3.13 ?
```

```text Bonne Requête
Quel est le prix actuel de l'action Tesla ?
```

```text Bonne Requête
Quelles sont les études les plus récentes sur le changement climatique de 2025 ?
```

</CodeGroup>

## Conseils

<Tip>
Tous les modèles dans Cattus ont cette capacité par défaut. Vous n'avez pas besoin de basculer vers un modèle spécifique ou d'activer des paramètres pour utiliser la recherche web.
</Tip>

<Info>
Le modèle déterminera quand la recherche web est nécessaire en fonction de votre requête. Pour les requêtes qui ne nécessitent pas d'informations actuelles, il peut s'appuyer sur ses données d'entraînement à la place.
</Info>

<Warning>
Bien que la recherche web donne accès à des informations actuelles, vérifiez toujours les informations critiques auprès de sources faisant autorité, en particulier pour les décisions importantes.
</Warning>
